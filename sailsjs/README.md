# 認識 Sails

![](assets/squiddy.png)

Sails 主要是使用 Express 的核心，再整合其他常用的 Express middleware 並搭配常用的模組，約定大於配置的目標為發展導向。所以我們只要使用 Sails.js 框架，即可達到簡單並快速的建構一個具有 MVC 架構的網站，內含的模組跟功能。

* socket.io
* 多種選配的資料庫 adapter
* 透過 cli 即可建立 REST API
* 內建基本的 grunt task
* (略...)
