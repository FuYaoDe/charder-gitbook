# CSS3

全名是 Cascading Style Sheets，簡寫 CSS

**CSS的歷史**

| 時間 | 事件     |
| :------------- | :------------- |
| 1996        | CSS1                                       |
|             | 背景、邊界、邊框、留白、定位、字型屬性、行距、對齊 |
| 1998        | CSS2                                       |
|             | 元素定位                                    |
|             | z 索引                                     |
|             | 字型陰影                                    |
|             | 雙向字型                                    |
| 2011        | CSS2.1                                     |
| 2012        | CSS3                                       |
|             | 媒體查詢                                    |
|             | 加強版選擇器                                 |
<!--
|             | 階層式(cascading)與繼承                      |
-->
|             | 模板佈局                                    |
|             | 轉換(transform)                             |
|             | 漸變(transition)                            |
