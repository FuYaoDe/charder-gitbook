# 環境安裝

### 必要條件

* ruby
* gem

### 安裝 SASS

```
gem install sass
```

或使用 GUI 工具

### 安裝 SCSS 版的 Bootstrap

* [Github - Bootstrap SCSS](https://github.com/twbs/bootstrap-sass)

```
npm i bootstrap-scss
```

### 轉換 SCSS 成 CSS

```
sass --watch input.sass:output.css
sass --watch input-dir:output-dir
```
