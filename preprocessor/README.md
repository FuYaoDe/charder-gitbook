# CSS 預處理器

預處理器可把含語法糖的程式碼，轉換成正常 CSS 格式。

**框架**

* SASS (Syntactically Awesome Stylesheets)
  * base on Ruby
  * 透過換行與縮排來表現這種層次結構
* SCSS（Sassy CSS）
  * base on Ruby
  * 從 SASS 進化而來
* LESS
* Stylus
  * base on NodeJS

  [sass_and_less_compared.markdown · GitHub](https://gist.github.com/chriseppstein/674726)
