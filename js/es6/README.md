# ES6

* Javascript 標準官方名稱是 [ECMAScript](https://en.wikipedia.org/wiki/ECMAScript) 簡稱 ES。
* ES6 (ES2015)，將慢慢成為主流。
* ES7 (ES2016)，在 2016 年底也已經 release 了。
