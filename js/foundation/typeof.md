# Typeof

typeof 運算子會傳回一個字串值, 指出未經運算 (unevaluated) 的運算元所代表的型別。

```js
var a = null;
console.log(typeof null);

var b;
console.log(typeof undefined);
```
<!-- object, undefined -->
