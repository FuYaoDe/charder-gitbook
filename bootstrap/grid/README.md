# 網格 (Grid)

![devices](http://getbootstrap.com/assets/img/devices.png)

以行動為優先的流動格線系統，並且隨著設備或可視區域大小縮放，格現系統會自動擴大最大至 12 個欄位。

### Grid 類別

| 標記名稱        | 銀幕尺寸           | 裝置  |
| :------------- | :--------------- | :---- |
| xs             | 768 px 以下       | 手機  |
| sm             | 768 px ~ 992 px  | 平板  |
| md             | 992 px ~ 1200 px | 桌機  |
| lg             | 1200 px 以上      | 投影機  |

### 練習題

* 將直式表單，換成可因應大小變成橫式表單。
